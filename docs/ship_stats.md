  Computed Stats (methods on Ship model)

  ┌───────────────────┬───────────────────────────────────────────────────────────┐
  │      Method       │                          Formula                          │
  ├───────────────────┼───────────────────────────────────────────────────────────┤
  │ getCombatRating() │ (hull_strength + shield_strength + weapon_slots * 50) / 3 │
  ├───────────────────┼───────────────────────────────────────────────────────────┤
  │ getUtilityScore() │ cargo_capacity + speed + utility_slots * 25               │
  └───────────────────┴───────────────────────────────────────────────────────────┘
