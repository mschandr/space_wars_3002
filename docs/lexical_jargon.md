**Lexical Jargon**

This file is to serve as documentation for all the lexical jargon within the 
game, Space Wars 3002 (at the moment this is strictly for the backend).

|       Term       |                    Meaning                    |                            Usage context                                               |
|------------------|-----------------------------------------------|----------------------------------------------------------------------------------------|
| Trading hub      | The place where all trade occurs              | This is traditionally used within star systems only.                                   |
| Trading station  | The Mineral marketplace                       | Here players trade minerals with merchants - the idea being you make a profit and can  |
|                  |                                               | buy better and better ships and other stuff.                                           |
| Shipyard         | This is the place where ship yards sell ships | This is again only available at star systems. Typically each star system  MAY have     |
|                  |                                               | one, and it should be one and only one.                                                |
| Salvage yard     | Where a player can get ship upgrades          | This is the place where a player can get ship upgrades for their CURRENT ship          |
| Current ship     | This is the ship is actively using at the time| The active ship can change for every request, so it's important to check the location  |
|                  |                                               | that the request and ship that it comes in for                                         |
| Star maps        | Player can consult for star charts            | Star charts are shared knowledge between all your ships. Therefore the star charts one |
|                  |                                               | gets, all of them get. The user can also sell stellar cartography maps to the player.  |
|                  |                                               | Once you sell a cartography station/store your star charts, anyone buying the star     |
|                  |                                               | charts from that point on, will get all the details they used to have + new details you|
|                  |                                               | just sold.                                                                             |
| Bar - rumours    | Player can use these places for information   | This is where a user might go to get rumours about pirates operating in the area - i.e.|
|                  |                                               | Potential price hikes, rumours about the precursor ships location is also possible, and|
|                  |                                               | lots of useful gossip about other players. Some bars may charge players to enter, those|
|                  |                                               | bars might provide better rumours - remember all rumours of the precursor ship are all |
|                  |                                               | wrong but the player has no idea of that fact.                                         |
| Bar - gambling   | Player can gamble a small amount of their     | Here the player can gamble a set amount of money - we will have "table limits" to      |
|                  | money to raise a bit of money.                | ensure they don't make too much money. Let's say no more than 10,000 credits can be    |
|                  |                                               | gambled before the establishment thinks you're cheating, throws you out and takes all  |
|                  |                                               | money you won, and 5% more to "teach you a lesson".                                    |
| Bar - board      | Jobs that players can pay to have completed   | This is another feature not yet implemented. The idea being after a certain point in   |
|                  |                                               | job it's actually more cost effective to pay other players to do certain things for you|
|                  |                                               | I haven't really quite planned this out yet.                                           |
| Bar - recruit    | Players can recruit crew, to enhance features | The idea that a good engineer, can get an extra 2% out of an engine, or think Scotty   |
|                  |                                               | from Star Trek or whatever. Or a weapons officer from some ship, people you can hire   |
|                  |                                               | Engineers - up to 5% improvement, Weapons officer - up to 5% improvement on targetting |
|                  |                                               | Reactor engineer - up to 5% fuel improvement, smuggler - can always hide an extra 10%  |
|                  |                                               | somewhere, scoundrel - up to 5% off cost on items.                                     |


Other things will be added later.
